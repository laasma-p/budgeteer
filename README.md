# budgeteer

budgeteer is an app that lets the user insert their expenses, item by item, see what products they exactly have bought and at which store the purchase has happened.

The motivation behind the idea is that I do not keep the receipts for a long use that I have collected at the store, but I keep a budget of where I can see on what I spend my money. However, it is done in an Microsoft Excel spreadsheet, which has turned out to be an outdated method for me, as well as when there are too many categories and data to be inserted, it becomes unreadable.

When it comes to the internet bank application, the data of what products I have exactly spent my money is not displayed there, only the categories and the shops where I have bought items that I need.

So far the app is intended only for private use, but if more people will show interest when, it might as well be refined for the future and possibly launched on Google Play Store.

### Requirements

- As a user, I want to be able to see the prices of products that I have bought so I can see on what I have spent my money on.
- As a user, I want to be able to make a weekly shopping list so I can see in one place what needs to be bought.
- As a user, I want to favorite the items that I buy often so I do not need to insert any data for the item every time I buy it.
- As a user, I want to be reminded that I need to purchase the things in the shopping list in case I forget so I would not have a day where I am missing a product that I really need to use.
- As a user, I want to be able to categorize the items bought so I can see where the money is spent.
- As a user, I want to be able to se the limit of how much money I can spend in a week so I can manage the expenses better.
